# The Young Raider

> **Survive, grow stronger, overcome relentless enemies, and climb the global ranks.**

**The Young Raider** is a fast-paced action-survival game developed for the **Reddit Devvit Hackathon**.

Built as an interactive Reddit experience, the game challenges players to control a Raider, battle increasingly dangerous enemies, destroy bases, survive devastating catastrophes, and compete for the highest score on a persistent global leaderboard.

Every run is a new opportunity to grow stronger, survive longer, surpass your previous record, and compete with other Reddit players.

## Play the Game

Play **The Young Raider** on Reddit:

[Play The Young Raider](https://www.reddit.com/r/TheYoungRaider/)

## About the Project

The Young Raider was created for the Reddit Devvit Hackathon using Reddit's interactive-post platform.

The project was adapted from a larger browser-game concept and redesigned to work as a short, replayable experience suitable for Reddit users on both desktop and mobile devices.

The game combines action combat, character progression, high-score competition, persistent leaderboard data, and community-focused features within a Reddit post.

Its main goals were to:

* Create gameplay that is easy to understand but difficult to master
* Encourage players to return and improve their personal best
* Support both desktop and mobile controls
* Integrate Reddit accounts directly into the leaderboard
* Use Devvit Redis for persistent progression and score storage
* Build community-oriented features around shared progress and competition

## Game Overview

Players begin each run with a basic Raider and limited combat statistics. By defeating enemies, collecting gold, purchasing upgrades, and destroying enemy bases, the Raider becomes progressively more powerful.

However, the battlefield becomes more dangerous over time. Enemies gain additional health, damage, and movement speed, while catastrophes and other threats force players to constantly move, adapt, and make strategic decisions.

The objective is:

> **Survive, conquer the battlefield, and achieve the highest score possible.**

## Main Features

### Fast-Paced Combat

Players have access to four core combat abilities:

* **Attack** — Perform a standard weapon attack.
* **Shield** — Block incoming attacks while the shield is raised in front of the player.
* **Slash** — Unleash a powerful close-range attack around the player.
* **Dash** — Dash forward before performing a sweep attack.

The game supports both desktop and mobile controls.

### Enemy Camps and Bases

Enemies defend camps and patrol the surrounding areas.

Destroying camps and bases rewards the player with valuable resources, but each victory also increases the difficulty of future encounters.

Newly generated enemy bases become progressively stronger, ensuring that every run continues to challenge the player.

### Score Multiplier

Each run begins with a high score multiplier that gradually decreases over time.

Players are encouraged to remain aggressive, defeat enemies quickly, and destroy objectives before the multiplier falls too far.

This creates a balance between playing safely and taking risks to achieve a higher score.

### Catastrophes

Powerful catastrophes periodically strike the battlefield.

Players must monitor the catastrophe timer, identify the danger area, and reposition before the attack lands.

Surviving these events becomes increasingly difficult as enemies continue to pressure the player.

### Enemy Strengthening

Enemies become stronger as the run progresses.

Their health, damage, and movement speed increase over time, with the current strengthening level displayed through the enemy strengthening timer and level indicator.

The longer the player survives, the more dangerous every enemy becomes.

### Battle Shop

Gold earned during a run can be spent on upgrades that improve the Raider's abilities.

Available upgrades include:

* Maximum health
* Attack damage
* Attack speed
* Movement speed
* Increased gold rewards
* Catastrophe immunity
* Additional powerful enhancements

Choosing the right upgrades is essential for surviving increasingly difficult enemies and extending the run.

### King Battles

Players can challenge a different King depending on the day.

King Battles feature:

* A gem entry cost
* Increasing difficulty after every victory
* Exclusive Raider rewards

Defeating a King can unlock an exclusive Raider for the player's collection.

### Raider Collection

Players can view, unlock, and select from a collection of Raiders.

Raiders may be unlocked through:

* High-score milestones
* Gem requirements
* King Battle victories

Each Raider has a unique appearance but stats remain same.

### Community Topics and Challenges

Players contribute their progress toward shared community goals.

Community challenge categories include:

* Damage
* Maximum health
* Starting gold

As the community reaches new milestones, permanent bonuses are awarded to all participating players.

These shared objectives allow every player's progress to contribute toward a larger collective achievement.

### Daily Rewards and Gem Requests

Players can claim daily gem rewards and request additional gems from the community.

They can also share their scores and leaderboard results with other players.

Gems can be used to:

* Enter King Battles
* Unlock selected Raiders
* Access special progression rewards

### Global Leaderboard

High scores are submitted to the global leaderboard.

Players can compare their best runs with other Raiders, compete for higher positions, and continue improving their personal records.

## Controls

### Desktop

| Input     | Action                            |
| --------- | --------------------------------- |
| **Mouse** | Move toward the selected location |
| **Q**     | Attack                            |
| **W**     | Shield                            |
| **E**     | Slash                             |
| **R**     | Dash                              |

### Mobile

| Input                | Action                                |
| -------------------- | ------------------------------------- |
| **Virtual joystick** | Move                                  |
| **Attack button**    | Perform a standard attack             |
| **Shield button**    | Raise or lower the shield             |
| **Slash button**     | Perform a powerful close-range attack |
| **Dash button**      | Quickly dash forward                  |

## Game Modes

### Normal Run

Survive for as long as possible while defeating enemies, destroying camps and bases, collecting resources, purchasing upgrades, and building a high score.

Enemy strength continuously increases, making every additional minute more challenging.

### King Battle

Challenge the current daily King in a dedicated battle.

Defeat increasingly powerful King levels to earn rewards and unlock exclusive Raiders.
